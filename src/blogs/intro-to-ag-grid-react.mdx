# Introduction to Ag-Grid: React

Ag-Grid is a fast, powerful, and feature-rich JavaScript grid library. Whether you're building an application with a simple table or a complex enterprise solution, Ag-Grid offers comprehensive customization with high performance. Ag-Grid comes in two versions: Community Edition and Enterprise Edition, with the latter requiring a per-developer license. The Enterprise Edition can be used without a license, but it will display a watermark that cannot be removed. (confirmed through testing).

Ag-Grid offers an extensive set of features, with key capabilities including:
-	High performance with large datasets
-	Seamless framework integration (React, Angular, Vue)
-	Column sorting with advance filtering
-	Pagination & infinite scroll (enterprise)
-	Customizable cell with flexible filtering options
-	Column resizing 
-	Advance search functionality
-	Row grouping and Aggregation
-	Data export to CSV (Community) and Excel (Enterprise)
